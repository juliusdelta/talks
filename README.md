# Talks

This is a repository of all my talks. I'm mostly using [MDX Deck](https://github.com/jxnblk/mdx-deck) to organize and do my slides.

## Local Viewing
- Clone the repo
- `cd` into the talk folder you want to view
- `npm install`
- `npm start`
